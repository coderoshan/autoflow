Frequently asked

How does AutoFlow extract tasks from Slack?

AutoFlow uses Groq AI to analyze messages in real-time. When you mention the bot or use natural language like "create a task to...", it automatically extracts the task name, priority, deadline, and category.

How does auto-delegation work?

Our AI analyzes each team member's skills, current workload, and past performance to recommend the best person for each task. You can approve or override the suggestion.

Is my data secure?

Yes. We use Supabase (PostgreSQL) with Row Level Security (RLS) enabled. Your Slack tokens are encrypted and never stored in plain text. We're SOC 2 compliant.

Can I use AutoFlow without Slack?

Currently, Slack is our primary integration. We're building support for Microsoft Teams, Discord, and standalone web input. Join the waitlist to get early access.
