# AutoFlow

AI-powered team automation app.

## Structure
- `bot/` - Node.js server (Slack, Zoom, AI)
- `mobile/` - React Native app
- `supabase/migrations/` - Database schema

## Setup
1. Run `npm install` in `bot/`
2. Copy `.env.example` to `.env` and fill in keys
3. Run `npm start` in `bot/`
4. Set up Supabase tables using migrations
